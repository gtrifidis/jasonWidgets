v2.1.0 2017 1st Release
New widgets

New features
	Use of font-awesome icons.
Updates
	Improved menu touch behavior.
	Improved grid column resizing.
	Improved grid column reordering.
Fixes
	Fixed bug with grid column reordering, where instead of a reorder a column swap was happening.

	Fixed bug with Menu and vertical orientation.
